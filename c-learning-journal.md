## 🧠 C语言考点清单（仅学考试需要的）

> 原则：够用就行，不贪多。每项能写出代码才算✅。

- [ ] 基本语法：数据类型、运算符、if/switch/for/while
- [ ] 函数：定义、调用、参数传递（**值传递 vs 地址传递**，408高频考点）
- [ ] 数组：一维/二维，与指针的关系（`a[i]` ≡ `*(a+i)`）
- [ ] **指针：`*` 和 `&`，指针与数组，指针与字符串，指向结构体的指针**
- [ ] **结构体：struct 定义、typedef、结构体指针（链表节点的基石）**
- [ ] **动态内存：malloc / free / sizeof（408综合题必涉及）**
- [ ] **递归：递归函数编写、递归转非递归（二叉树/排序必用）**
- [ ] 字符串：char 数组、`strlen/strcpy/strcmp`用法
- [ ] 文件操作：`fopen/fclose/fscanf/fprintf`（复旦机考可能要求文件读写）
- [ ] 宏定义：`#define`、`#include`（基本够用即可）

## 📖 408数据结构 — 七章考点逐个击破

### 第1章 绪论
- [ ] 数据结构三要素（逻辑/存储/运算）
- [ ] 算法五特性
- [ ] **时间复杂度 & 空间复杂度分析**（每年必考，会算O(n)）

### 第2章 线性表
- [ ] 顺序表：定义、插入、删除、查找（`O(1)` 随机存取）
- [ ] 单链表：头插法、尾插法、按值查找、删除节点
- [ ] 双向链表：插入、删除
- [ ] 顺序表 vs 链表的选用场景（408选择题高频）
- [ ] 🔨 **手写实现**：顺序表增删改查、单链表增删改查

### 第3章 栈、队列和数组
- [ ] 栈：顺序栈、链栈、入栈出栈
- [ ] 队列：顺序队列、**循环队列**（判空/判满条件，必考）
- [ ] 链式队列
- [ ] 栈的应用：括号匹配、表达式求值
- [ ] 特殊矩阵压缩存储：对称矩阵、三角矩阵、稀疏矩阵
- [ ] 🔨 **手写实现**：顺序栈、循环队列

### 第4章 树与二叉树 ⭐重中之重
- [ ] 二叉树5条性质（记死，选择题秒杀用）
- [ ] 二叉树存储：顺序存储、链式存储（`lchild`/`rchild`）
- [ ] **四种遍历**：前序、中序、后序、层序（递归 + 非递归）
- [ ] **线索二叉树**：概念与构造
- [ ] 树↔森林↔二叉树 相互转换
- [ ] **哈夫曼树 & 哈夫曼编码**（构造 + WPL计算，必考）
- [ ] 并查集
- [ ] **堆**（2026考纲新增）
- [ ] 🔨 **手写实现**：二叉链表建树、四种遍历、哈夫曼树构建

### 第5章 图 ⭐高频
- [ ] 存储：**邻接矩阵**、**邻接表**（两种都要会写代码）
- [ ] 遍历：**DFS**（递归+栈）、**BFS**（队列）
- [ ] **最小生成树**：Prim、Kruskal
- [ ] **最短路径**：Dijkstra、Floyd
- [ ] **拓扑排序**（AOV网）
- [ ] 关键路径（AOE网）
- [ ] 🔨 **手写实现**：邻接表建图、DFS/BFS、Dijkstra

### 第6章 查找
- [ ] 顺序查找、分块查找、**折半查找**（判定树）
- [ ] **二叉搜索树 BST**（插入、删除、查找）
- [ ] **平衡二叉树 AVL**（四种旋转：LL/RR/LR/RL）
- [ ] 红黑树（五条性质，选择题常考）
- [ ] **B树 & B+树**（结构区别，每年几乎必考）
- [ ] **哈希表**（哈希函数 + 开放定址法/链地址法冲突处理）
- [ ] 字符串模式匹配（2026考纲新增）
- [ ] 🔨 **手写实现**：BST增删查、AVL旋转、哈希表

### 第7章 排序 ⭐核心
- [ ] **快速排序**（枢轴划分 + 递归，最常考）
- [ ] **堆排序**（建堆 + 调整堆）
- [ ] **归并排序**（二路归并）
- [ ] 插入排序 / 折半插入排序
- [ ] 冒泡排序
- [ ] 简单选择排序
- [ ] 希尔排序
- [ ] 基数排序
- [ ] 外部排序（多路归并）
- [ ] **必须能闭卷默写**：各排序时空复杂度 + 稳定性表格
- [ ] 🔨 **手写实现**：快排、堆排、归并排序（三道必能手写）
